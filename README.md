# Binance Project Crawler

## Installation

1. Install [NodeJS](https://nodejs.org/en/)
2. Install yarn, by running `npm i -g yarn`
3. Install packages, `yarn`

## Usage

```
Usage: crawler aave-protocol

{
  tokenSummary: {
    holdersMakeMoney: {
      in: 22,
      ...
    }
  }
}
```

## Testing
